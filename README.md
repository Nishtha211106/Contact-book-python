# Contact Book CLI App — Python
A command line contact management application with persistent storage.
Features:

Add contacts with duplicate detection
View contacts alphabetically with total count
Search by name or phone number
Update existing contacts
Delete contacts
Phone number validation (10 digits only)
Persistent JSON file storage

Concepts Used:

File handling (JSON)
Error handling (try/except)
List comprehensions
f-strings
Lambda functions
Password hashing (hashlib SHA256)

How to run:

Clone the repository
Run python contact_book.py
Set your password on first run
